# 古诗词学习网

一个基于 Vue3 的古诗词学习网站，采用新中式水墨风格设计，提供诗词浏览、搜索、文言文词典查询等功能。

## 项目简介

这是一个人机交互课程项目，专注于良好的用户体验设计和界面构思。项目采用 Vue3 + Vite 构建，无需后端支持，所有数据存储在本地。

## 功能特性

### 🏠 首页
- 顶层分类导航（唐诗、宋词、元曲、先秦散文）
- 全局搜索功能
- 热门诗词推荐

### 📚 诗词库
- 诗词列表浏览
- 分类筛选
- 搜索功能
- 诗词详情页（正文、注释、赏析）
- 收藏功能

### 📖 文言文词典
- 文言文字词查询
- 详细释义展示
- 例句和出处
- 查询历史记录

### 👤 我的
- 我的收藏管理
- 学习轨迹统计
- 个人资料编辑
- 设置（主题切换、字体大小）
- 关于我们和反馈

## 技术栈

- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **路由**: Vue Router 4
- **样式**: CSS3 (新中式水墨风格)

## 安装和运行

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

项目将在 `http://localhost:3000` 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 设计特色

### 新中式水墨风格
- **色彩体系**: 深绿色（#2E463B）作为主色调，米白色（#FAF9F6）作为背景
- **字体**: 宋体，传递古典雅致感
- **背景**: 淡墨晕染纹理，营造留白意境
- **交互**: 卡片悬浮效果、页面切换动画、聚焦反馈

### 用户体验优化
- 响应式设计，适配不同屏幕尺寸
- 流畅的页面切换动画
- 直观的导航和搜索功能
- 本地存储，保存用户偏好和历史记录

## 数据说明

项目使用示例数据，所有诗词和词典数据存储在 `src/utils/data.js` 中。实际项目中可以：
- 集成真实的后端 API
- 使用本地 JSON 文件
- 连接数据库

## 开发说明

### 添加新诗词
在 `src/utils/data.js` 的 `samplePoems` 数组中添加新的诗词对象。

### 添加词典词条
在 `src/utils/data.js` 的 `dictionaryData` 对象中添加新的词条。

### 自定义样式
修改 `src/styles/main.css` 中的 CSS 变量来调整主题色彩。
